# Streamlit-Saree-Draping-App

# 👗 AI Saree Draping App

An AI-powered virtual saree draping application that allows users to visualize how different saree styles look on them. Built using **Streamlit**, this project provides an interactive and simple interface for experimenting with saree draping styles digita

## 📌 Features

* 🧍 Upload your image
* 👗 Try different saree draping styles virtually
* ⚡ Fast and lightweight UI using Streamlit
* 🤖 AI-based visualization concept
* 📱 Beginner-friendly interface

---

## 🛠️ Tech Stack

* **Frontend/UI:** Streamlit
* **Backend:** Python
* **Libraries:**

  * OpenCV
  * PIL

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/BurukalaManiReethika/Streamlit-Saree-Draping-App.git
cd Streamlit-Saree-Draping-App
```

### 2️⃣ Create virtual environment (optional but recommended)

```bash
python -m venv venv
source venv/bin/activate   # for Linux/Mac
venv\Scripts\activate      # for Windows
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the app

```bash
streamlit run app.py
```
